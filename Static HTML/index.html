Key Features

Multiple Header

Adora includes 3 different header designs to choose from. It also has sticky header option as well. You can switch elements in the header section, change color, add your own logo.

Mega Menu

Adora has a MegaMenu built into the theme which is perfect for large menus. You can place page links, images, maps, forms and more. Each menu can have custom icon next to it.

Mobile Responsive

Adora is 100% responsive, each and every element including the awesome sliders are fully responsive. It utilizes the power of Bootstap 4 to make everything truly responsive.

2000+ Icon

Adora uses Fontawesome icons, 1000+ custom line icons and 1000+ custom flat icons. Using font icons speeds up the load times and ensures super crisp on all screen size.

SEO Optimized

Adora is ready to get you higher rankings because it is built with SEO best practices in mind. It uses valid HTML code and CSS which allows search engines to easily index you site.

Ecommerce Ready

Adora is 100% ready for ecommerce website, it also includes full design integration that looks amazing! Check demo to see how beautiful your shop can be with Adora.
